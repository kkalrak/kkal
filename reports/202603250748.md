## 1. 나프타란 무엇인가?

### 정의

```
나프타 (Naphtha / 납사):
├─ 원유 정제 시 추출되는 가벼운 액체 탄화수소
├─ 끓는점: 30~200°C 구간
├─ 구성: C5~C10 저분자 탄화수소 혼합물
└─ 별명: "석유화학의 쌀"
```

### 용도

```
나프타 → 크래킹(열분해) → 기초 화학원료

에틸렌:
├─ 플라스틱 (PE, PVC)
├─ 합성섬유 (폴리에스터)
└─ 세제, 부동액

프로필렌:
├─ 폴리프로필렌 (PP)
├─ 아크릴
└─ 페인트 원료

기타:
├─ 합성고무
├─ 비료 원료
└─ 의약품 원료
```

**일상 속 나프타 관련 제품:**
```
├─ 플라스틱 포장재 (모든 식품 포장)
├─ 페트병
├─ 합성섬유 의류 (폴리에스터, 나일론)
├─ 타이어 (합성고무)
├─ 가전제품 외장재
├─ 의약품 캡슐
└─ 비료 (농업)
```

## 2. 한국의 나프타 공급 구조

### 수입 의존도

```
한국 나프타 수입 현황:
├─ 연간 수입량: 약 3,000만 톤 이상
├─ 중동 의존도: 70%+ (사우디, UAE, 쿠웨이트, 이란)
├─ 아시아산: 15% (말레이시아, 인도네시아)
└─ 기타: 15% (북해, 미국)

핵심 경로:
중동 산유국 → 호르무즈 해협 → 한국 정유사 → 나프타 생산
```

### 공급망 구조

```
원유 수입
   ↓
국내 정유사 (GS칼텍스, SK에너지, 에쓰오일, HD현대오일뱅크)
   ↓
정제 → 나프타 추출
   ↓
석유화학사에 공급
   ↓
LG화학, 롯데케미칼, 한화솔루션, 금호석유화학
   ↓
에틸렌, 프로필렌 등 기초 화학원료 생산
   ↓
중간재, 완성품 제조
```

## 3. 이란 전쟁의 나프타 충격

### 위협 요인 3가지

**1️⃣ 호르무즈 해협 봉쇄 위협 (가장 위험)**

```
이란의 위협 (3월 23일):
"미국이 발전소를 치면
 페르시아만 전체에 기뢰를 매설하겠다"

호르무즈 해협의 의미:
├─ 세계 원유 거래량의 30% 통과
├─ 한국 원유 수입의 30% 통과
├─ 폭: 가장 좁은 곳 불과 21km
└─ 기뢰 매설 시: 사실상 통행 불가

봉쇄 시 영향:
├─ 원유 공급 급감 → 나프타 생산 급감
├─ 나프타 가격: 현재 대비 50-100% 상승 가능
└─ 한국 석유화학: 즉시 원가 압박
```

**2️⃣ 이란산 원유 추가 제재**

```
이란의 위상:
├─ 세계 4위 원유 매장량 (1,578억 배럴)
├─ 세계 7위 원유 생산국
└─ 중질유 → 나프타 생산 효율 높음

현재 제재 상황:
├─ 기존 미국 제재: 이미 진행 중
├─ 트럼프 강화 가능성: 높음
└─ 제재 확대 시: 글로벌 공급 더 감소
```

**3️⃣ 유가 급등 연동**

```
유가 현황:
├─ 현재: $110-130 (급등 중)
├─ IEA 경고: "1970년대 이후 최악"
├─ 나프타 가격: 유가에 연동

계산:
원유 $100 시 나프타: 약 $700/톤
원유 $130 시 나프타: 약 $900/톤
원유 $150 시 나프타: 약 $1,050/톤+

한국 연간 수입 3,000만 톤 기준:
유가 $30 상승 → 연간 추가 비용 약 $6B (약 9조원)
```

```
├─ 배터리 부문은 영향 적음

├─ 합성고무 (타이어 원료) 생산

```

```
├─ 석유화학 부문: 타격

SKC:
├─ 필름, 화학 소재
└─ 원료 비용 상승

대한유화:
├─ 일부 LPG 크래킹 가능 (대체 원료)

```

**🟢 상대적 수혜:**

```

├─ 정유사 → 나프타 판매자
├─ 나프타 가격 상승 = 판매 수익 증가
├─ 사우디아람코 계열 → 공급 안정
└─ 수혜 기업 ⭐

GS칼텍스:
├─ 정유 마진 개선
```

## 5. 파급 효과: 일상으로의 전이

### 물가 상승 경로

```
나프타 가격 상승
   ↓
에틸렌/프로필렌 가격 상승
   ↓
플라스틱 원재료 (PE, PP, PVC) 가격 상승
   ↓
포장재 비용 증가
   ↓
식품, 생활용품 가격 상승
   ↓
소비자 물가 (CPI) 상승

예상 파급 규모:
나프타 50% 상승 → 소비재 물가 +1.5-2%
나프타 100% 상승 → 소비재 물가 +3-4%
```

### 산업별 2차 타격

```
자동차 산업:
├─ 타이어 원가 상승 (합성고무)
├─ 내장재 비용 증가 (플라스틱)
└─ 차량 가격 인상 압박

식품 산업:
├─ 포장재 비용 급증
├─ 용기, 봉지 모두 나프타 기반
└─ 식품 가격 인상

건설 산업:
├─ PVC 파이프, 단열재 가격 상승
└─ 건축 원가 증가

농업:
├─ 비료 원료 가격 상승
├─ 농업 생산비 증가
└─ 식품 물가 2차 상승
```
