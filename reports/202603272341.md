# 이란 전쟁 → 나프타 부족 → 쓰레기봉투 품귀: 공급망 연결 분석

**작성일**: 2026년 3월 27일 23시 41분  
**주제**: 중동 전쟁이 한국 일상 소비재에 미치는 직접적 영향  
**분류**: 거시경제 & 중동 분석  

---

## 📊 Executive Summary

```
사실 확인: ✅ 연결되어 있습니다

이란 전쟁 → 나프타 공급 차질
         → 폴리에틸렌(PE) 생산 감소
         → 비닐·쓰레기봉투 품귀 현실화

현재 상황: 품귀 시작 단계 (3월 27일 기준)
핵심 변수: 호르무즈 해협 봉쇄 여부
```

## 1. 공급망 연결 구조

```
이란 전쟁
   ↓
호르무즈 해협 위기 (기뢰 매설 위협)
   ↓
중동산 원유·나프타 공급 감소 + 가격 급등
   ↓
나프타 크래킹(열분해) 공정 감소
   ↓
에틸렌 생산 감소
   ↓
폴리에틸렌(PE) 생산 감소
   ↓
비닐봉투·쓰레기봉투 원료 부족
   ↓
가격 인상 + 공급 품귀
```

---

## 2. 쓰레기봉투의 원료: 왜 나프타가 핵심인가

```
쓰레기봉투 원료:
├─ LDPE (저밀도 폴리에틸렌): 얇고 유연한 봉투
└─ HDPE (고밀도 폴리에틸렌): 두껍고 단단한 봉투

LDPE·HDPE 생산 과정:
나프타 → 크래킹 → 에틸렌 → 중합 → 폴리에틸렌(PE)

한국의 PE 생산:
├─ LG화학, 롯데케미칼, 한화솔루션 등
├─ 원료의 70%+가 중동산 나프타
└─ 나프타 가격 30% 상승 → PE 가격 25~30% 상승
```

**실제 뉴스 (3월 25~26일):**

```
영문 보도:
├─ "이란 전쟁으로 나프타 공급 차질
│   → 한국 비닐 제품 부족 우려 확산"
│   (Fears of plastic bag shortage in South Korea)
│
├─ "한국 시장에서 비닐봉투 품귀 현상"
│   (Middle East conflict causes plastic bag
│    shortages in South Korea)
│
└─ "나프타 병목 → 폴리에틸렌 가격 급등
    → 봉투류 시장에서 사라지는 중"
    (Market bags run short)
```

**현재 실제 영향:**

```
소비자:
├─ 대형마트: 봉투 재고 감소 시작
├─ 편의점: 비닐봉투 수급 불안
└─ 온라인: 봉투류 가격 인상 시작

제조사:
├─ 소규모 봉투 제조사: 원료 수급 어려움
├─ PE 원료 가격: 급등 중
└─ 생산 비용 증가 → 출고가 인상

유통:
├─ 지자체 종량제봉투: 아직 정상
└─ 시중 비닐류: 품귀 신호 감지
```

## 4. 영향 범위: 봉투만이 아니다

```
나프타 → PE → 영향받는 제품 전체:

일상 소비재:
├─ 쓰레기봉투 (종량제 포함)
├─ 마트·편의점 비닐봉투
├─ 포장용 랩·지퍼백
├─ 택배 포장재 (에어캡, 완충재)
└─ 식품 포장 필름

농업:
├─ 하우스 비닐 (농산물 가격 간접 영향)
└─ 멀칭 필름

산업:
├─ 공장 포장재
├─ 건설용 방수 필름
└─ 의료용 포장재

→ 물가 전반에 영향
```

## 5. 얼마나 심각한가?

**현재~단기 (1개월 이내):**

```
상황: 품귀 시작 단계
비축: 국내 약 30~40일치 재고 보유
가격: 인상 중이지만 구할 수는 있음
위기: 초기 경보 수준

호르무즈 봉쇄 없으면:
└─ 가격 높지만 수급 유지 가능
```

**중기 (1~3개월): 호르무즈 봉쇄 시**

```
나프타 공급 급감 (30%+)
   ↓
PE 생산 대폭 감소
   ↓
봉투류 생산 감소 (30~50%)
   ↓
진짜 품귀 현상
   ↓
가격 2~3배 급등 가능
```

**비교: 2021년 반도체 대란과 유사**

```
2021년: 코로나 → 공장 셧다운 → 반도체 부족
        → 자동차 생산 중단 → 차값 급등

2026년: 이란 전쟁 → 나프타 부족 → PE 부족
        → 봉투·포장재 품귀 → 소비재 가격 급등

패턴: 원자재 충격 → 공급망 → 소비재
```

## 6. 정부 대응 가능성

```
가능한 정책:

1️⃣ 나프타 비축량 방출
   └─ 정부 전략 비축유에서 나프타 방출
      효과: 단기 공급 완화

2️⃣ PE 수입선 다변화 긴급 추진
   └─ 미국산, 북해산 PE 긴급 수입
      효과: 중간 정도

3️⃣ 봉투류 가격 상한제
   └─ 소비자 부담 완화
      효과: 제한적 (생산비 해결 안 됨)

4️⃣ 기름값 상한제 연계
   └─ 유류세 인하로 원가 부담 일부 완화
      효과: 간접적

한계:
└─ 근본 원인(이란 전쟁)이 해결되지 않으면
   임시방편에 불과
```

## 7. 소비자 행동 가이드

```
지금 당장:
├─ 종량제봉투: 여유분 확보 권장
│  (가격 인상 전 미리 구매)
├─ 비닐랩·지퍼백: 소량 비축
└─ 포장재 절약 습관 시작

주의:
├─ 사재기 금지 (공황 구매는 품귀 악화)
└─ 정부 대응 지켜보기

대안:
├─ 천 장바구니 활용
├─ 다회용 용기 사용 확대
└─ 음식물 쓰레기 비닐 절약
```
