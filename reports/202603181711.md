## 📋 Executive Summary (업데이트)

고려아연은 **기술력은 세계 최고**이지만, **재무 위기 + 지정학적 위험 + 투자 부담**이라는 **3중 위험**에 직면해 있습니다. 특히 이란 전쟁으로 인한 공급망 불안정과 미국/중국 간의 경제 갈등이 직접적 위험요인입니다.

## 1. 지정학적 위험 분석

### 1-1. 이란 전쟁과 아연 공급망 위기 🚨

**현재 상황:**
```
이란 전쟁 → 중동 항로 불안정 → 아연 광석 수송 차질
→ 중국 제련소 공급 부족 → 아연 가격 변동성 증가
```

**공급망 구조:**
- 글로벌 아연 원료: 페루, 호주, 중국, 인도 등에서 생산
- 중국: 세계 아연의 35% 이상 생산 (최대 생산국)
- 중동 항로: 중국으로 가는 주요 무역로
- **문제**: 이란 전쟁으로 탱커/화물선 우회 운항 → 비용 증가, 지연

**고려아연의 영향:**

| 시나리오 | 확률 | 영향도 | 설명 |
|---------|------|-------|------|
| **아연 가격 ↑ 10-15%** | 40% | +++ | 매출 증가, 영업이익 개선 |
| **공급 부족 지속** | 50% | +++ | 수요 초과, 가격 상승 우호적 |
| **수출 차질** | 20% | --- | 판매 애로, 채산성 악화 |
| **전쟁 장기화** | 60% | --- | 불확실성 증가, 투자 부진 |

**단기 효과 (2026년):**
- ✅ 긍정: 아연 가격 상승 가능 → 매출 증가
- ❌ 부정: 수송비 증가, 불확실성으로 수요 둔화

**결론:**
```
단기: 긍정과 부정의 혼재 (불확실성 높음)
장기: 공급망 재편, 비용 증가 가능성
```

### 3-3. ② 그린수소 (Green Hydrogen)

**개념:**
```
신재생에너지(태양광·풍력)
→ 전기 생산
→ 물(H₂O) 전기분해
→ 수소(H₂) 생산 ← "그린수소"
→ 제련 공정 에너지원으로 사용
```

**도입 일정:**
- 초기 단계: 2025-2027년 소규모 시험
- 본격화: 2028년 이후
- 목표: 2030년대 주요 에너지원

**투자 규모:**
- 추정: 1조원대 이상
- 시간: 장기 (10년 이상)

**수익성:**
```
2026-2029년: 손실만 발생 (투자 단계)
2030년 이후: 비용 절감으로 수익 기여
```

**시장 전망:**
```
글로벌 그린수소 시장:
- 2025년: 500억 달러
- 2030년: 1,000억 달러 (2배)
- 2040년: 2,000억 달러 (4배)

성장성: ⭐⭐⭐⭐⭐ (5/5)
현재 수익 기여: ⭐ (1/5)
```

**위험 요소:**
- 아직 초기 기술, 상용화 불확실
- 정부 정책 변동 위험
- 경제성 미확보 상태

**평가:**
```
⭐⭐⭐ (3/5)
미래가치는 높지만 현재는 비용만 증가
```

### 3-4. ③ 자원순환 (Circular Economy)

**① 폐플라스틱 재활용**
- **BOTTLE 컨소시엄과 협력**
  - 폐플라스틱을 화학원료로 재자원화
  - 고려아연: 제련 기술 활용한 화학 처리
- **시장 규모**: 연간 수십억 달러 (성장 중)
- **현황**: 2025-2026 파일럿 단계
- **상용화**: 2027-2028년 예상

**② 폐자원 회수**
- **부산물로부터 귀금속 회수**
  - 아연 제련 부산물 → 금·은·구리 추출
  - 부가 수익 창출
- **현황**: 이미 일부 시행 중
- **확대**: 기술 고도화로 수익 증대

**③ 전략광물 생산 (가장 주목)**
- **고부가 광물:**
  - 인듐 (Indium): 반도체, 터치스크린 → 가격 kg당 수십만원
  - 게르마늄 (Germanium): 광학, 반도체 → 가격 kg당 수천~만원
  - 갈륨 (Gallium): 반도체, 태양광 → 가격 kg당 수백만원
  
- **비교:**
  ```
  아연 가격: kg당 2,000-3,000원
  인듐 가격: kg당 500,000-1,000,000원 (500배!)
  ```

- **시장 전망:**
  ```
  EV 배터리 수요 증가
  → 반도체, 태양광 수요 급증
  → 전략광물 수급 부족
  → 가격 상승, 공급 우대
  ```

- **현황:**
  - 아직 초기 단계 (전체 수익의 5% 미만)
  - 2025-2027년 생산 증대
  - 2028년 이후 수익 가시화 예상

**자원순환의 평가:**
```
⭐⭐⭐⭐⭐ (5/5) - 가장 유망한 사업

장점:
✅ 이미 일부 수익화 중
✅ 전략광물의 미래가치 높음
✅ 환경 친화적 (ESG)
✅ 중국 의존도 감소

단점:
❌ 아직 초기 단계
❌ 생산량 증대 시간 필요
❌ 기술 개발 투자 필요

결론: 3-5년 후 고수익원 가능성
```

```

   → 중국 과잉생산 심화

```
