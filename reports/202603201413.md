**분류**: 거시경제 & 미국 경제

## 📊 Executive Summary

**국방비 지출 현황:**
```
초기 3개월 추가 지출: 약 $3-5B

장기 (1년+): 부정 (누적 부채 문제)
```

## 1. 국방비 지출의 경제학: 두 가지 관점

### 🟢 긍정적 관점: 케인즈 승수 효과

**메커니즘:**

```
정부 국방 지출 증가
  ↓
방위산업 회사들의 매출 증가 (Lockheed, Raytheon 등)
  ↓
기업 이윤 증가 → 배당금, 주가 상승
  ↓
고용 확대 (직접/간접)
  ├─ 직접: 방위산업 노동자
  ├─ 간접: 공급사, 운송업, 서비스업
  └─ 고용 증가: 약 10만-20만 명
  ↓
가계 소득 증가 → 소비 증가
  ↓
소비 증가 → 다른 산업 매출 증가
  ↓
GDP 성장 (+0.3-0.5%)
```

**승수 효과 (Multiplier Effect):**

```
정부 지출: $1
  ↓
경제에 미치는 총 효과: $1.5-2.0
  (각 부문의 추가 지출로 증폭)

예시:
$1B 국방 지출
  → 직접 고용 임금: $300M
  → 이들의 소비: $250M
  → 다른 업체 매출 증가: $150M
  → 추가 고용 및 소비: $100M
  → ... (계속 증폭)
  ↓
총 경제 효과: $1.5-2.0B
```

**현재 지출의 승수:**

```
초기 지출: $3-5B
승수율: 1.5-1.8x (추정)
총 경제 효과: $4.5-9.0B

GDP 성장 기여도:
$4.5-9.0B ÷ $28조 GDP = +0.16% ~ +0.32%
```

### 🔴 부정적 관점: 재정 악화와 구축효과

**메커니즘:**

```
정부 국방 지출 증가
  ↓
재정 적자 확대
  (이미 $1.8조 적자 상태에서 추가)
  ↓
정부 차입금 증가 (국채 발행)
  ↓
국채 발행 증가 → 금리 상승 압박
  ├─ 10년물 수익률: 4.2% → 4.5-4.7%
  └─ 이유: 공급 증가로 가격 하락
  ↓
금리 상승 → 기업/가계 차입 비용 증가
  ├─ 기업: 투자 감소 (설비, R&D)
  ├─ 가계: 주택 구매 감소, 소비 감소
  └─ 결과: 민간 투자 위축
  ↓
구축효과 (Crowding Out):
정부 지출 ↑ → 민간 투자 ↓
(한정된 자본이 정부에 쏠림)
  ↓
경제 성장 둔화 (-0.1% ~ -0.3%)
```

**구축효과의 크기:**

```
정부 지출 증가: $3-5B
금리 상승폭: 0.2-0.3%
(국채 수익률 0.2-0.3% 상승)

금리 상승으로 인한 투자 감소:
$10B 투자 × 0.25% = $25M 감소 (연간)
100배 확대하면 → $2.5B 투자 감소

결과:
GDP 성장 감소: -0.09% ~ -0.15%
```

## 2. 순 효과 계산: 승수 효과 vs 구축효과

### 이론적 계산

```
케인즈 승수 효과:     +0.16% ~ +0.32% (GDP)
구축효과:           -0.09% ~ -0.15% (GDP)
─────────────────────────────────
순 효과:             +0.01% ~ +0.17% (GDP)

평가:
└─ 소폭의 긍정 (또는 중립)
└─ 거의 무시할 수 있는 수준
```

### 실제 효과 (현실적 계산)

```
추가 변수들:

1️⃣ 유가 상승 동시 진행
   └─ 인플레이션 추가 압박
   └─ 실질 경제 둔화

2️⃣ 이미 높은 실업률
   └─ 케인즈 승수 약함
   └─ 추가 고용 효과 제한적

3️⃣ 금리 이미 높음
   └─ 구축효과 크게 작용
   └─ 민간 투자 대체 어려움

4️⃣ 소비자 신뢰도 약화
   └─ 추가 소비 의욕 낮음
   └─ 승수 약화

종합 평가:
└─ 순 효과: 거의 중립 (-0.05% ~ +0.05%)
└─ 경제 성장에 유의미한 영향 거의 없음
```

## 3. 시간대별 경제 영향

### 단기 (1-3개월): 약한 긍정

**발생 상황:**
```
현재 상태 (2026년 3월)
├─ 지출 공시
├─ 주문 증가
└─ 기업들 즉각적 반응 시작
```

**경제 효과:**
```
즉각적 효과:
├─ 방위산업 주가 상승 (+5-10%)
├─ 기업 매출 호재 기대
├─ 고용 증가 신호
└─ 소비심리 약간 개선

GDP 기여:
└─ +0.2-0.3% (단기 부분적 반영)

평가:
└─ "좋은 뉴스지만 시간차 있음"
```

### 중기 (3-6개월): 중립~약한 부정

**발생 상황:**
```
지출 반영 시작
├─ 실제 고용 증가
├─ 기업 이윤 반영
├─ 금리 상승 본격화
└─ 민간 투자 위축
```

**경제 효과:**
```
케인즈 효과:
├─ 고용 약 5-10만 명 창출
├─ 임금 소비 증가
└─ GDP +0.2-0.3% 기여

구축효과:
├─ 금리 0.2-0.3% 상승
├─ 기업 투자 감소
├─ 주택 시장 약화
└─ GDP -0.15-0.25% 감소

순 효과:
└─ -0.05% ~ +0.05% (거의 중립)

다른 요인들:
├─ 유가 여전히 높음
├─ 인플레이션 지속
├─ 실업 증가 신호
└─ 경제 약세 기조 지속
```

**시장 반응:**
```
주식:
├─ 방위산업: +3-5% (추가 상승)
├─ 일반 기업: -1-2% (구축효과)
└─ S&P 500: -0.5% ~ +0.5%

채권:
├─ 10년물 수익률: 4.4-4.6% (상승)
├─ 채권 가격: -2-3% (하락)
└─ 장기 채권 약세

달러:
├─ 금리 상승 → 달러 강세
├─ USD/EUR: 1.08-1.10
└─ 수출 경쟁력 약화
```

### 장기 (6-12개월+): 부정적

**발생 상황:**
```
누적 부채 효과 본격화
├─ 국가 부채 계속 증가
├─ 금리 고착
├─ 민간 투자 지속 위축
└─ 성장력 약화
```

**경제 효과:**

```
문제점 심화:
├─ 국가 부채: 이미 GDP의 125%
├─ 추가 지출로 130%+ 올라갈 우려
├─ 구조적 인플레이션 우려
└─ 장기 금리 상승 추세

경제 영향:
├─ GDP 성장: 1.5-2.0% (둔화 지속)
├─ 실업률: 4.5-5.5% (상승)
├─ 인플레이션: 3.0-3.5% (높음)
└─ 주가: -5-10% (조정)

장기 위험:
└─ 부채 악순환 시작 가능성
```

```
방위산업 채용:
├─ Lockheed Martin: +3,000-5,000명
├─ Raytheon: +2,000-3,000명
├─ General Dynamics: +1,500-2,500명
├─ 기타 방위업체: +3,000-5,000명
└─ 총 직접: 약 10,000-15,000명

간접 고용:
├─ 공급사 채용: +3,000-5,000명
├─ 운송, 로지스틱: +2,000-3,000명
├─ 서비스업: +2,000-3,000명
└─ 총 간접: 약 7,000-11,000명

전체 고용 증가:
└─ 약 17,000-26,000명 (추정)
```

**고용의 질:**

```
방위산업 일자리 특징:

└─ "좋은 일자리" 창출

```

```
방위산업 임금 인상:

└─ 인플레이션 추가 요인

정부 일자리:
├─ 방위산업 지원 일자리
├─ 임금 인상 기대
└─ 총 임금 수준 +1-2%

평가:
└─ 노동자에게 긍정적
└─ 기업 비용 증가 (음수 효과)
```

## 6. 재정 및 금리에 미치는 영향

### 정부 재정 상태 악화

**현황:**

```
2026년 예상 재정 상황:

세입: 약 $4.5조
세출: 약 $6.3조 (기존)
기존 적자: 약 $1.8조

추가 국방비: $8-12B
새로운 세출: 약 $6.31조
새로운 적자: 약 $1.81조

변화:
┌─────────────────┬──────────┬──────────┐
│ 항목            │ 기존     │ 추가비용 │
├─────────────────┼──────────┼──────────┤
│ 국방비          │ $820B    │ +$10B    │
│ 적자            │ $1.8T    │ +$10B    │
│ 부채 (GDP대비)  │ 125%     │ +0.04pp  │
└─────────────────┴──────────┴──────────┘
```

**장기 문제:**

```
누적 부채:
├─ 2026년 말: 약 $33.5조
├─ GDP 대비: 약 125%+
└─ 선진국 중 높은 수준

문제점:
├─ 세대 간 부채 이전
├─ 인상분 급증 가능성
├─ 금융 위기 위험 증가
└─ 통화 정책 제약
```

### 금리에 미치는 영향

**메커니즘:**

```
정부 국채 발행 증가
  ↓
국채 공급 증가
  ↓
국채 가격 하락 (공급 > 수요)
  ↓
국채 수익률 상승
  ↓
전체 금리 수준 상승
```

**수익률 변화 예측:**

```
현재: 약 4.2-4.3%
추가 발행 시: 4.4-4.6%

상승폭: 0.2-0.3%

영향 기간:
├─ 즉시: 약간의 상승
├─ 3개월 후: 본격적 상승
├─ 6개월 후: 안정화
└─ 1년 후: 새로운 "정상" 수준

영향 받는 분야:
├─ 주택 담보 대출금리: +0.2-0.3%
├─ 자동차 할부금리: +0.15-0.25%
├─ 기업 차입금리: +0.2-0.3%
└─ 신용카드 금리: +0.25-0.4%
```

## 7. 인플레이션에 미치는 영향

```

  ↓
수요 증가 (방위산업)
  ↓
그에 따른 물가 상승
  ↓
기대 인플레이션 상승

크기:
└─ 약 0.1-0.2% (작은 수준)
```

### 간접적 효과 (금리 상승)

```
금리 상승
  ↓
기업 원가 증가
  ↓
제품 가격 인상
  ↓
인플레이션 상승

크기:
└─ 약 0.15-0.25% (중간 수준)
```

### 유가 상승과의 복합 효과

```
유가 상승: +0.5% (이미 분석됨)
국방비 지출: +0.15-0.25%
─────────────────────
총 인플레이션: +0.65-0.75%

누적:
├─ 기존 예상: 2.8-3.2%
├─ 추가: +0.65%
└─ 새로운 예상: 3.45-3.85%

## 8. 투자자 전략

### 섹터별 투자 전략

**방위산업 (Overweight):**

```
매수 대상:
├─ Lockheed Martin (LMT)
├─ Raytheon (RTX)
├─ General Dynamics (GD)
└─ Northrop Grumman (NOC)

이유:
├─ 직접 수혜 기업
├─ 단기 주가 상승 기대
├─ 이윤 확대

기간:
├─ 단기 (3-6개월): +5-10%
├─ 중기 (6-12개월): +3-7%
└─ 평가: 호재 지속 기간까지

추가:
