## 1. 한국의 호르무즈 해협 현황

### 현재 배치 현황

**배치 내용:**
```
함정: 1척 (정기 순환)
├─ 호위함 (Frigate급)
├─ 규모: 2,500톤급
└─ 인원: 약 150-200명

지원함: 1척 (보급함)
├─ 유조선형 보급함
└─ 인원: 약 100-150명

총 규모:
├─ 함정 2척
├─ 인원: 약 300명
└─ 파견 기간: 12개월 (정기 순환)
```

**참여 체계:**
```
다국적군 (CMF: Coalition Maritime Forces)
├─ 미국 주도
├─ 연합국 함대 (5-6개국)
├─ 한국도 정규 멤버
└─ 주 역할: 해상 감시, 해적 소탕
```

### 파견의 중요성

**에너지 안보:**
```
한국의 석유 수입:
├─ 총 수입: 약 100만 배럴/일
├─ 중동산: 약 65-70%
├─ 호르무즈 통과: 약 30%
└─ 금액: 연간 약 $30-40B

의존도: 높음 (⭐⭐⭐⭐⭐)
```

**경제 이익:**
```
해운 시장:
├─ 한국 선사: 8-10개
├─ 호르무즈 통과 선박: 월 200-300척
├─ 해상 보험료: 
│  - 평시: $50-100/컨테이너
│  - 전쟁 시: $300-500+ (5-10배)
└─ 한국 경제 피해: 매우 큼
```

## 3. 한국이 추가 파견할 가능성

### 파견 시나리오

**시나리오: 1척만 계속 유지 (확률 65-70%)**

```
상황:
├─ 현재 상황 지속 (긴장 높음, 전쟁 아님)
├─ 이란 보복 유보 상태
└─ 호르무즈 통행 계속 가능

한국의 대응:
├─ 기존 1척만 정기 순환
├─ 추가 파견 없음
├─ 미국과는 계속 협력
└─ 이란과는 경제 관계 유지

근거:
✅ 에너지 안보 필요 (1척은 필수)
✅ 미국 동맹 의무 (1척으로 충분)
❌ 추가 파견은 이란 관계 악화
❌ 한국 여론 (추가 파견 반대)

확률: 65-70% (가장 높음)
```

### 추가 파견이 없는 이유

**경제적 고려:**
```
한국-이란 관계:
├─ 석유 수입 (연간 $10-15B)
├─ 건설 수주 (대우건설, 현대 등)
├─ 정유 산업 투자
└─ 중요도: 매우 높음

위험:
└─ 추가 파견 → 이란 경제 보복 가능
   (석유 공급 감소, 계약 취소 등)
```

**외교적 고려:**
```
한국의 입장:
├─ "미국과도 친하고, 중국과도 친하고"
├─ "중동 국가들과도 좋은 관계"
└─ 목표: 최대한 중립적 입장 유지

추가 파견의 위험:
├─ 이란: "너희는 미국 편이군" → 보복
├─ 중국: "미국에 너무 친해" → 압박
├─ 아랍권: "너희도 기독교 국가?" → 의심
└─ 결과: 한국 외교 신뢰도 하락
```

**여론상 고려:**
```
국내 의견:
├─ "미국 때문에 전쟁에 참가하나?" (부정적)
├─ "석유 때문에 파견한다" (비판)
└─ "한국은 중립이어야" (강함)

정치적 위험:
├─ 2026년 총선 (4월 1주)
├─ 야당의 비판 가능
└─ 여론 악화 → 정부 신뢰도 하락
```

## 4. 한국이 파견해야 하는 이유

### 필수 파견 근거

**에너지 안보 (가장 중요):**
```
호르무즈 해협의 의미:
├─ 세계 석유 30% 통과 지점
├─ 한국은 특히 의존도 높음 (30%)
├─ 통행 불능 = 한국 경제 마비 수준

한국의 선택지:
├─ 1척 파견: "호르무즈 안전에 동참"
├─ 파견 거부: "우리 에너지 보호 못 한다"
└─ 결론: 파견은 필수

비용 편익:
├─ 파견 비용: 연 $100-150M
├─ 석유 안정 수익: 연 $5-10B+
└─ 비율: 1 : 50 이상 (매우 유리)
```

**미국 동맹 (중요):**
```
미국의 입장:
├─ "해상 안전은 모두의 책임"
├─ "선진국들은 다 참여"
└─ "한국도 당연히 응해야"

한국의 이익:
├─ 한미동맹 강화
├─ 국제적 신뢰도 상승
└─ 향후 한국이 위기 시 미국 지원받을 때 "신뢰의 자산"

비용:
└─ 연 $100-150M (무시할 수 없지만 필요)
```

**국제 책임:**
```
한국의 위상:
├─ G20 멤버 (세계 11위 경제)
├─ 안보리 비상임이사국 경험
├─ "선진국"으로서의 책임
└─ "해상 안보 협력은 당연"
```

### 가장 가능성 높은 시나리오

```
✅ 기존 1척만 계속 배치 (65-70% 확률)

이유:
1️⃣ 에너지 안보 필수
   └─ 호르무즈 해협 통행 확보 필요

2️⃣ 미국 동맹 의무
   └─ 1척은 "최소한의 역할"

3️⃣ 이란 관계 유지
   └─ 석유, 건설 계약 보호

4️⃣ 여론 관리
   └─ "추가 파견은 없다"는 메시지
```

### 한국의 전략

```
1️⃣ 외교 전선
   ├─ 미국: "1척으로 최선을 다하겠다"
   ├─ 이란: "우리는 중립입니다"
   └─ 중동 국가: "한국은 신뢰할 수 있습니다"

2️⃣ 경제 전선
   ├─ 석유 장기 계약 확보 (사우디, UAE)
   ├─ 해운 보험 협상 (보험료 최소화)
   └─ 건설 계약 유지 (이란과)

3️⃣ 안보 전선
   ├─ 1척 배치 유지
   ├─ 전자전 감시 강화
   └─ 군인 안전 최우선

4️⃣ 국내 정치
   ├─ "1척으로 충분"이라는 설명
   ├─ "이란과 관계는 좋다"는 설명
   └─ 여론 관리 (총선 대비)
```
