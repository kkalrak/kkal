# 드론 격추 비용 전쟁: CIWS-II vs 레이저 vs 미사일 완전 비교

**작성일**: 2026년 3월 26일 11시 04분  
**주제**: 현대 방공 무기 3종 비용·성능·미래 분석  
**분류**: 한국 방산 & 안보  

---

## 📊 Executive Summary

```
드론 1대 격추 비용:

미사일:   $3,000,000+ (43억원+)  ← 너무 비쌈
CIWS-II:  $40,000~80,000 (6~12천만원)
레이저:   $1~수십달러 (1,500원~수만원) ← 혁명적
```

**결론: 레이저가 경제성 압도. 단, 날씨 한계 있어 CIWS-II와 조합 필수.**

## 5. 레이저의 치명적 약점

```
1️⃣ 날씨: 안개·비·모래폭풍 → 출력 급감
   (호르무즈 해협 = 모래폭풍 지역)

2️⃣ 표적 강화 가능: 반사 코팅 드론 개발 시 무력화

3️⃣ 1회에 1표적: 드론 100대 동시 → 하나씩만 격추

4️⃣ 전력 의존: 150kW = 함정 전력에 큰 부하
```

## 6. 비대칭 전쟁의 경제학

```
이란 드론 1대 생산: $20,000 (약 3,000만원)
CIWS-II로 격추:    $40,000~80,000

"드론이 더 싸다" → 방어자가 항상 손해

레이저 등장 후:
이란 드론 1대 생산: $20,000
레이저로 격추:      $1~수십달러

→ 공격자가 손해로 역전
→ 드론 공격의 경제적 우위 消滅
```

## 7. 미래 함정 방공망 최적 조합

```
다층 방공 구성:

원거리 (100km+): 미사일 (SM-2, SM-6)
근거리 (2~10km): 레이저 — 드론 떼 1차 처리
초근접 (0~4km):  CIWS-II — 레이저 뚫은 것 최후 격추

역할 분담:
├─ 레이저: 날씨 좋을 때 경제적으로 처리
├─ CIWS-II: 악천후·레이저 실패 시 백업
└─ 미사일: 크루즈 미사일 등 고위협 표적만

"레이저 + CIWS-II 조합이 미래 표준"
```
