# CIWS-II vs 레이저 방공: 킬체인·탐지·격추 기술 심층 비교

**작성일**: 2026년 3월 26일 11시 14분  
**주제**: 근접방어무기와 레이저의 핵심 기술 차이 분석  
**분류**: 한국 방산 & 안보  

---

## 📊 Executive Summary

```
CIWS-II: 운동에너지(탄환) 방식 → 전천후, 즉각적, 검증된 기술
레이저:  열에너지(광자) 방식  → 경제적, 정밀하나 날씨·재질에 취약

현재: CIWS-II 우위 (신뢰성)
미래: 레이저 주력 + CIWS-II 백업
```

---

## 1. 킬체인 비교 (탐지→추적→격추)

### CIWS-II

```
① 탐지: Ka밴드 레이더 (밀리미터파)
   └─ 전파 반사 기반, 초당 수백 회 스캔

② 추적: 폐루프(Closed-Loop) 방식
   └─ 발사한 탄환 궤적까지 추적
      → 탄과 표적 편차 실시간 자동 보정
      → "탄이 어디 갔는지 보면서 수정"

③ 격추: 30mm 텅스텐 관통자
   └─ 충격(운동에너지)으로 즉시 파괴

총 킬 타임: 약 3~5초
```

### 레이저 (HELIOS)

```
① 탐지: EO/IR + 레이더 융합
   └─ 열신호 추적 → 스텔스도 열 나면 탐지

② 추적: 고정밀 짐벌 마운트 (빔 디렉터)
   └─ 조준 정밀도: 마이크로라디안 수준
      → 초당 수십 회 위치 자동 보정

③ 격추: 파이버 레이저 결합
   └─ 여러 빔을 하나로 합쳐 출력 집중
      → 열에너지로 표적 표면 용융·폭발

총 킬 타임: 약 4~12초 (표적 재질에 따라)
```

---

## 2. 물리적 격추 원리

```
CIWS-II (운동에너지):
탄환 → 고속 충돌(1,000m/s+) → 충격 파괴
└─ 표적 크기 무관, 산탄 효과, 즉시 파괴

레이저 (열에너지):
광자 → 표면 흡수 → 가열 → 용융/폭발
└─ 수 초~수십 초 조사 필요 (표적에 머물러야 함)

속도 의외성:
"드론 격추는 CIWS-II가 더 빠를 수 있다"
└─ 레이저는 얇은 재질도 수 초 조사 필요
```

## 3. 핵심 기술 비교표

```
┌────────────────────────────────────────────────┐
│              CIWS-II        레이저              │
├────────────────────────────────────────────────┤
│ 탐지방식   레이더 중심     EO/IR+레이더 융합    │
│ 추적방식   폐루프 탄도보정 고정밀 짐벌          │
│ 격추원리   운동에너지      열에너지             │
│ 반응속도   3~5초           4~12초               │
│ 전천후     ✅ 강함         ❌ 악천후 취약       │
│ 다중표적   ⭐⭐⭐ 가능     ⭐ 1회 1표적         │
│ 스텔스대응 ❌ 약함         ✅ 열신호 탐지       │
│ 전력소모   낮음(수kW)      높음(150kW+)         │
│ 신뢰성     ✅ 검증됨       ⚠️ 신기술            │
└────────────────────────────────────────────────┘
```

## 4. 표적별 효과

```
소형 드론:
├─ CIWS-II: ✅ 용이 (1~2발로 파괴)
└─ 레이저:  ✅ 용이 (얇은 재질, 수 초 소손)

크루즈 미사일:
├─ CIWS-II: ✅ 강함 (고속 연사로 격추)
└─ 레이저:  ⚠️ 어려움 (두꺼운 금속, 조사시간 부족)

드론 떼 (10대+):
├─ CIWS-II: ⚠️ 과부하 위험
└─ 레이저:  ⚠️ 하나씩만 처리 가능

탄도미사일:
├─ CIWS-II: ❌ 속도 초과
└─ 레이저:  ❌ 속도 초과
```

## 5. 전자전(ECM) 저항성

```
CIWS-II:
├─ 레이더 재밍: 취약할 수 있음
└─ 단, 탄환 자체는 재밍 불가
   "발사 후엔 재밍 무의미"

레이저:
├─ 재밍: 광학 센서로 보완 가능 (유리)
├─ 반사 코팅: 레이저 반사로 무력화 가능
└─ 연막탄: 산란으로 효과 감소

핵심: 적의 물리적 대응(코팅, 연막) = 레이저 약점
```

---

## 6. 한국 CIWS-II의 기술적 차별점 (팰렁스 대비)

```
① 구경 20mm → 30mm: 파괴력 2배+, 사거리 40% ↑

② 탐지: 레이더 단독 → 레이더 + EO/IR 융합

③ 추적: 단일 표적 → 다중 표적 동시 추적

④ 탄약: APDS + 프로그래머블 신관
   └─ 공중 근접 폭발 가능 → 드론 대응 최적화

⑤ AI 기반 표적 분류: 아군 오인 격추 방지
```

## 7. 전력·냉각 문제

```
CIWS-II:
├─ 전력: 수 kW (함정 부담 적음)
└─ 냉각: 공냉 (단순)

레이저 150kW:
├─ 전력: 함정 전체 전력의 10~20%
├─ 냉각: 수냉 필수 (복잡)
└─ 실전 문제:
   "레이저 쏠 때 레이더·엔진 출력 낮춰야"
   → 함정 설계 단계부터 통합 필요
```

## 8. 미래 발전 방향

```
CIWS-II:
├─ 레일건 방식 탄약 검토 (극초음속)
├─ AI 자율교전 고도화
└─ 소형화 → 차량·기지 배치용

레이저:
├─ 출력: 150kW → 300kW → 1MW
├─ 다중빔: 여러 표적 동시 격추 (연구 중)
└─ 대기 보정: 악천후 극복 기술 개발 중

10년 후:
"레이저가 주력, CIWS-II는 악천후·백업"
```
