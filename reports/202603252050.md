# 천무2 vs CIWS-II: 한국 미래 방공망의 두 축

**작성일**: 2026년 3월 25일 20시 50분  
**주제**: 한국 차세대 방공 무기 체계 비교 및 역할 분석  
**분류**: 한국 방산 & 안보  

---

## 📊 Executive Summary

```
천무2: 공격으로 방어하는 "화력의 왕" (장거리 정밀타격)
CIWS-II: 뚫고 들어온 것을 막는 "방패의 왕" (초근접 요격)

드론 전쟁 시대: CIWS-II의 가치가 급상승 중
미래 방공망: 두 무기의 조합이 완성형
```

---

## 1. 두 무기의 핵심 차이

```
┌─────────────────────────────────────┐
│           천무2          CIWS-II    │
├─────────────────────────────────────┤
│ 역할   공격적 방어      수동적 방어  │
│ 사거리 80~290km        2~4km        │
│ 플랫폼 지상 차량       함선 탑재    │
│ 대상   지상 목표       비행체 요격  │
│ 방공   적 발사대 제거  미사일·드론  │
└─────────────────────────────────────┘
```

**천무2의 방공 철학:**
```
"적이 쏘기 전에 발사대를 먼저 부순다"
→ 공격으로 방어 (선제 제압)
```

**CIWS-II의 방공 철학:**
```
"어떻게든 날아온 것은 내가 막는다"
→ 최후 방어선 (근접 요격)
```

---

## 2. CIWS-II 제원 및 성능

```
구경: 30mm (기존 팰렁스 20mm → 업그레이드)
분당 발사: 수천 발 (자동)
사거리: 2~4km
반응시간: 수 초 이내 (완전 자동)

핵심 개선사항 (팰렁스 대비):
├─ 구경 30mm → 파괴력·사거리 증가
├─ 광전자 추적 장치 통합 (레이더+카메라 융합)
├─ 다중 표적 동시 추적·격추 설계
└─ 드론 떼 공격 대응 최적화
```

---

## 3. 드론 전쟁 시대: CIWS-II가 빛나는 이유

**현재 전장의 변화:**
```
이란-이스라엘: 드론 떼 공격 일상화
우크라이나: 드론 수천 대 동시 운용
미래: 드론 포화 공격 (Swarm) 대세화
```

**기존 팰렁스의 한계:**
```
팰렁스: 한 번에 1개 표적 추적
드론 10대+ 동시 돌진 → 과부하 → 격추 실패
```

**CIWS-II의 해법:**
```
다중 표적 동시 추적·격추
→ 드론 떼 방어 가능성 대폭 향상
→ 이란전쟁·우크라이나가 이미 필요성 증명
```

---

## 4. 천무2와 CIWS-II의 조합: 완전한 방공망

**이스라엘 방식 참고:**
```
Arrow:      탄도미사일 (장거리)
David's Sling: 중거리 요격
Iron Dome:  드론·로켓 근접 요격
→ 3단 다층 방공망 = 세계 최고 수준
```

**한국의 미래 방공망 (예상):**
```
고도·거리별 다층 구조:

장거리 (100km+):
└─ 패트리어트 PAC-3, THAAD, 천궁-II

중거리 (10~100km):
└─ 천궁, 신궁

단거리 공격 (80~290km):
└─ 천무2 ← 적 발사대 선제 제거

초근접 (0~4km):
└─ CIWS-II ← 뚫고 들어온 것 최후 격추

결론: 천무2 + CIWS-II = 상호 보완적 완성형
```

---

## 5. CIWS-II의 미래 가능성

**육상 배치 확대 가능성:**
```
현재: 함선 탑재 전용

미래:
├─ 주요 군사기지 근접방어
├─ 항구, 공항, 핵발전소 방어
├─ 드론 공격 노출된 전방 시설
└─ → 함선 밖으로 역할 확대 가능
```

**수출 시장:**
```
글로벌 팰렁스 교체 시장:
├─ 운용국: 15개국 이상
├─ 드론 위협 급증으로 교체 수요 폭증
└─ CIWS-II 수출 성공 시: 수조원 시장

관련 기업:
├─ LIG넥스원 (주관 개발사)
├─ 한화시스템 (레이더·전자전)
└─ 현대중공업 (함정 통합)
```

## 6. 투자 관점

```
방산주 수혜 기업:

LIG넥스원:
├─ CIWS-II 주관 개발사
├─ 천궁, 신궁도 보유
└─ 드론 방어 시장 확대 = 직접 수혜

한화에어로스페이스:
├─ 천무 계열 생산
├─ K9 자주포 등 수출 호조
└─ 방공 수요 확대 수혜

한화시스템:
├─ 레이더·전자전 전문
├─ CIWS-II 레이더 연동
└─ 방공 통합 솔루션

모니터링 지표:
├─ 방위사업청 CIWS-II 전력화 일정 발표
├─ 수출 계약 발표 (수출 성공 시 주가 급등)
└─ 이란전쟁 장기화 → 방산 예산 증가
```
