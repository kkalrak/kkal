# 깔깔 주식 보고서 - 작성 가이드라인

## 목표
투자자가 참고할 수 있는 **신뢰성 높은 주식 분석 보고서** 작성

---

## 보고서 구조

### 1. 헤더 정보 (필수)
```
# [회사명] 주식 분석 보고서

**작성일**: YYYY년 MM월 DD일 HH시 MM분  
**종목코드**: XXXXXX  
**회사명**: 회사 정식명칭
```

### 2. 본문 섹션 (필수 순서)

#### 1. 기업 개요
- 회사 설립연도, 대표이사
- 주요 사업 부문 (불릿 포인트)
- 회사의 위치/규모 (시가총액 등)

#### 2. 최근 실적 분석
- **표 형식으로** 매출, 영업이익, 순이익 비교
- 연도별 또는 분기별 비교
- 변화율(%) 표시

#### 3. 강점 분석
- ✅ 3~5개의 강점
- 각 강점마다 1~2줄 설명

#### 4. 약점 및 위험요인
- ⚠️ 3~5개의 리스크
- 각 위험요인마다 1~2줄 설명

#### 5. 투자 의견
- **목표가**: 숫자 입력
- **추천 등급**: BUY / HOLD / SELL 중 선택
- 의견 요약 (1~2문단)

### 3. 이미지 포함 (필수)

**2장의 이미지 포함 방식:**

```markdown
![회사 로고](image1.jpg)
![재무 그래프](image2.jpg)
```

**이미지 저장 위치:** `reports/images/{YYYYMMDDHHmm}/`

**이미지 요구사항:**
- 크기: 800x600px 이상
- 포맷: JPG 또는 PNG
- 용도:
  - 이미지1: 회사 로고 또는 주요 제품/서비스
  - 이미지2: 재무 차트, 그래프, 또는 기업 사진

## 자료 수집 방법

### 1. 검색할 정보
- **회사 기본정보**: 공식 홈페이지, 위키백과
- **재무정보**: 한국거래소(KRX), IR 공시, 최신 뉴스
- **경영현황**: 최근 3개월 뉴스
- **산업분석**: 관련 시장 보고서

### 2. 신뢰성 있는 정보원
✅ **신뢰도 높음**
- 한국거래소 (KRX)
- 회사 공식 IR 페이지
- 언론사 경제 섹션 (조선일보, 동아일보, 한경 등)
- 애널리스트 보고서

⚠️ **참고만 (검증 필요)**
- 개인 블로그
- 유튜브 재테크 채널
- 커뮤니티 (주식 커뮤니티 등)

### 3. 이미지 수집
- **로고**: 공식 홈페이지에서 직접 다운로드
- **그래프**: 공시자료, 뉴스 기사에서 스크린샷
- **기업사진**: 공식 뉴스룸, 보도자료

## 마크다운 작성 규칙

### 제목 사용
```
# H1 - 보고서 제목 (1개만)
## H2 - 주요 섹션 (기업개요, 실적분석 등)
### H3 - 소제목 (필요 시)
```

### 표 작성 (실적 데이터)
```markdown
| 구분 | 2024년 | 2025년 | 변화 |
|------|--------|--------|------|
| 매출액 | 금액 | 금액 | +% |
| 영업이익 | 금액 | 금액 | +% |
```

### 강점/약점 마크
```markdown
✅ 강점 항목
⚠️ 위험요인
```

### 인용문
```markdown
> "인용 내용 또는 강조하고 싶은 문구"
```

| 항목 | 기준 |
|------|------|

| 이미지 | 정확히 2장 |
| 정확성 | 최신 공시 정보 기반 |
| 객관성 | 개인 의견이 아닌 시장 분석 기반 |

## 파일명 규칙

**파일명**: `YYYYMMDDHHmm.md`

예시:
- `202603171215.md` (2026년 3월 17일 12시 15분)
- `202603171300.md` (2026년 3월 17일 13시 00분)
